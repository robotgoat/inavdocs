## Gyroscope Calibration

:::info
Under normal conditions there is no need for a manual calibration procedure, but if required this can be performed via stick commands.
:::

Gyroscope calibration, or rather bias recording, is performed on every startup.
**Your model should be stationary while powering up.**

With some models, connecting the battery while keeping the craft still can be difficult. 
Ensure the craft is placed on the ground (or somewhere solid and still) for 5 seconds as soon as possible after powering up.
**Gyro auto calibration will only run when no motion is detected.**
